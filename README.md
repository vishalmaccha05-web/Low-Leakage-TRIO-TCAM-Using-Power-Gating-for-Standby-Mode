# Low-Leakage-TRIO-TCAM-Using-Power-Gating-for-Standby-Mode
The Low-Leakage TRIO-TCAM Using Power Gating project aims to reduce standby leakage power in TCAM used for high-speed search applications. It uses a compact 10-transistor TRIO-TCAM cell and a dual-sleep power-gating technique to disconnect power during idle mode, improving energy efficiency while maintaining fast search performance.
